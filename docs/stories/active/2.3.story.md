# Story 2.3: Análise Farmacoterapêutica Completa

**Epic:** 2 — Farmacoterapia
**Status:** Ready
**Prioridade:** P0
**FR:** FR-9
**Estimativa:** L (6-8h)

## Descrição

Como farmacêutico, preciso realizar e registrar a análise farmacoterapêutica completa do paciente, avaliando interações medicamentosas, duplicidade terapêutica, adequação da dose à condição clínica e adesão ao tratamento, com campo para observações detalhadas.

## Contexto Técnico

Já existe `/api/analises` com integração Claude AI básica. Esta story EXPANDE esse módulo para o padrão clínico completo, adicionando os critérios estruturados de análise (interação, duplicidade, dose, condição).

## Acceptance Criteria

- [ ] AC-1: Avaliação estruturada por critério — cada item com status (Adequado / Inadequado / Não avaliado) e observação:
  - Interações medicamentosas
  - Duplicidade terapêutica
  - Adequação da dose
  - Adequação à condição clínica
  - Adesão ao tratamento — se avaliação de adesão (Story 2.5) disponível para o atendimento, exibir taxa(s) calculada(s) como evidência objetiva no critério
- [ ] AC-2: Campo de observações gerais (texto rico)
- [ ] AC-3: Análise associada ao atendimento (não apenas ao paciente)
- [ ] AC-4: Geração assistida por IA (Claude) pré-preenche a análise com base nos medicamentos e dados clínicos registrados
- [ ] AC-5: Farmacêutico pode editar/complementar o que a IA sugeriu antes de salvar
- [ ] AC-6: Histórico de análises por paciente com data e farmacêutico responsável

## Escopo

**IN:**
- Expandir model `AnalyseFarmaco` existente com critérios estruturados
- Manter integração Claude AI com contexto clínico enriquecido
- Associar análise ao atendimento

**OUT:**
- Verificação de interações via base de dados dedicada (DrugBank/Micromedex) — v2

## Valor de Negócio

Análise formal documentada é o diferencial do serviço farmacêutico clínico e requisito para faturamento de serviços. Integração com IA (Claude) reduz tempo de análise de ~45min para ~5min, aumentando o throughput do farmacêutico e viabilizando atendimento de mais pacientes.

## Riscos

1. Contexto de IA pode ser insuficiente se medicamentos e dados clínicos não estiverem cadastrados — exibir aviso ao usuário quando contexto estiver incompleto.
2. Expansão do model `AnalyseFarmaco` precisa ser retrocompatível com dados existentes — adicionar campos novos como nullable.

## Definition of Done

- Critérios estruturados (interação, duplicidade, dose, condição, adesão) salvos por análise
- IA gera análise pre-preenchida com base em medicamentos e dados clínicos do paciente
- Farmacêutico pode editar sugestões da IA antes de salvar
- Histórico de análises visível no prontuário com data e farmacêutico responsável
- Migration retrocompatível aplicada sem erros
