# Story 2.1: Resultados de Exames Laboratoriais

**Epic:** 2 — Farmacoterapia
**Status:** Draft
**Prioridade:** P0
**FR:** FR-7
**Estimativa:** M (4h)

## Descrição

Como farmacêutico, preciso registrar e visualizar os resultados de exames laboratoriais do paciente (glicemia de jejum, HbA1c, colesterol, creatinina, TFG), com referência de valores normais, para orientar a análise farmacoterapêutica.

## Acceptance Criteria

- [ ] AC-1: Exames padrão pré-definidos: Glicemia de Jejum, Hemoglobina Glicada (HbA1c), Colesterol Total, LDL, HDL, Triglicerídeos, Creatinina, Taxa de Filtração Glomerular (TFG), outros (texto livre)
- [ ] AC-2: Para cada exame: valor, unidade, data de coleta, laboratório (opcional), valor de referência (informativo)
- [ ] AC-3: Alerta visual se valor fora da faixa de referência
- [ ] AC-4: Possibilidade de anexar PDF/imagem do laudo (referência à Story 4.1)
- [ ] AC-5: Histórico de exames por tipo (evolução ao longo do tempo)
- [ ] AC-6: Exames associados ao paciente (não necessariamente ao atendimento)

## Escopo

**IN:**
- Model `ResultadoExame` no Prisma
- CRUD de exames laboratoriais
- Valores de referência embutidos no código
- Histórico por tipo de exame

**OUT:**
- Integração com laboratórios via HL7/FHIR (v3)
- Interpretação automática via IA (coberta na Story 4.2)
