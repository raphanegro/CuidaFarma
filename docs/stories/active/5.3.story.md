# Story 5.3: Relatório e Exportação

**Epic:** 5 — Relatórios e Gestão
**Status:** Draft
**Prioridade:** P0
**FR:** FR-28
**Estimativa:** M (4h)

## Descrição

Como farmacêutico, preciso exportar o prontuário completo do paciente e os dados de atendimentos em PDF, para compartilhar com outros profissionais de saúde ou arquivar fisicamente.

## Acceptance Criteria

- [ ] AC-1: Exportação do prontuário completo do paciente em PDF (dados pessoais + histórico clínico + medicamentos + análises + intervenções)
- [ ] AC-2: Exportação de atendimento específico em PDF
- [ ] AC-3: Exportação da lista de pacientes em CSV (dados básicos)
- [ ] AC-4: PDF com logo/header do sistema, data de geração e farmacêutico responsável
- [ ] AC-5: Opção de imprimir diretamente pelo browser

## Escopo

**IN:**
- PDF de prontuário completo
- PDF de atendimento
- CSV de lista de pacientes
- Impressão browser

**OUT:**
- Exportação para Word/Excel (v2)
- Template personalizável de PDF (v2)

## Dependências

- Dados dos Epics 1-3 consolidados

## Tasks de Implementação

- [ ] Integrar biblioteca PDF: `@react-pdf/renderer` ou `jsPDF` + `html2canvas`
- [ ] Criar template de prontuário PDF
- [ ] Criar API `/api/relatorios/prontuario/[pacienteId]` que gera o PDF
- [ ] Criar API `/api/relatorios/pacientes.csv` para exportação CSV
- [ ] Botões de exportar/imprimir nas páginas de prontuário e listagem
