# Story 3.3: Estratificação de Risco

**Epic:** 3 — Intervenção e Acompanhamento
**Status:** Draft
**Prioridade:** P0
**FR:** FR-16
**Estimativa:** M (3-4h)

## Descrição

Como farmacêutico, preciso classificar o paciente em nível de risco (baixo, moderado, alto) com base em critérios clínicos automatizados e julgamento clínico, para priorizar os casos que necessitam de maior atenção.

## Acceptance Criteria

- [ ] AC-1: Critérios automáticos de risco avaliados:
  - Polifarmácia: ≥5 medicamentos = risco moderado, ≥10 = alto
  - Idade: ≥65 anos = +1 nível de risco
  - Doenças crônicas: ≥3 = moderado, ≥5 = alto
  - PRMs identificados com gravidade grave = alto automaticamente
  - Baixa adesão registrada = +1 nível (usa última taxa de adesão da Story 2.5 para o paciente; se ausente, critério não é pontuado automaticamente)
- [ ] AC-2: Farmacêutico pode ajustar a classificação manualmente com justificativa
- [ ] AC-3: Nível de risco exibido em destaque no prontuário (badge colorido: verde/amarelo/vermelho)
- [ ] AC-4: Histórico de estratificações por paciente
- [ ] AC-5: Dashboard mostra distribuição de pacientes por nível de risco

## Escopo

**IN:**

- Exibição visual no prontuário
- Ajuste manual com justificativa

**OUT:**
