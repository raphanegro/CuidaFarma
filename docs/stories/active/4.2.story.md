# Story 4.2: Sugestões do Assistente Clínico

**Epic:** 4 — Assistente Clínico e Documentos
**Status:** Draft
**Prioridade:** P0
**FR:** FR-13
**Estimativa:** L (6-8h)

## Descrição

Como farmacêutico, preciso que o sistema gere alertas automáticos sobre interações medicamentosas, doses inadequadas, riscos clínicos e alterações de exames, com sugestões de intervenções farmacêuticas baseadas no perfil completo do paciente.

## Acceptance Criteria

- [ ] AC-1: Alertas gerados ao abrir o prontuário ou ao salvar novos dados clínicos:
  - Interações medicamentosas (baseado nos medicamentos em uso)
  - Doses acima/abaixo da faixa recomendada
  - Exames fora da faixa de referência
  - Polifarmácia (≥5 medicamentos)
  - Medicamento sem indicação registrada
- [ ] AC-2: Cada alerta tem: tipo, severidade (Informativo/Atenção/Crítico), descrição, medicamento(s) envolvido(s), sugestão de ação
- [ ] AC-3: Alertas críticos exibidos com destaque visual no topo do prontuário
- [ ] AC-4: Farmacêutico pode marcar alerta como "visto" ou "ignorado com justificativa"
- [ ] AC-5: Sugestões de intervenção geradas pelo Claude com base nos alertas e histórico clínico
- [ ] AC-6: Sugestões pré-preenchem o form de intervenção (Story 3.1) — farmacêutico edita antes de salvar

## Escopo

**IN:**
- Análise baseada nos dados já registrados (medicamentos em uso, exames, dados clínicos)
- Integração com Claude AI para sugestões de intervenção contextualizadas
- Alertas em tempo real ao carregar prontuário

**OUT:**
- Base de dados de interações medicamentosas (DrugBank/Micromedex) — usar Claude como proxy no MVP
- Monitoramento contínuo em background (v2)
