# Story 3.2: Calendário Posológico

**Epic:** 3 — Intervenção e Acompanhamento
**Status:** Draft
**Prioridade:** P0
**FR:** FR-15
**Estimativa:** L (6-8h)

## Descrição

Como farmacêutico, preciso gerar automaticamente o calendário posológico do paciente com base nos medicamentos em uso cadastrados, com possibilidade de editar horários, imprimir e exportar como PDF.

## Acceptance Criteria

- [ ] AC-1: Geração automática do calendário a partir dos medicamentos em uso ativos (Story 2.2)
- [ ] AC-2: Exibição em formato de grade: colunas = horários do dia (manhã, tarde, noite, madrugada), linhas = medicamentos
- [ ] AC-3: Farmacêutico pode editar horários individualmente após geração automática
- [ ] AC-4: Opção de imprimir diretamente pelo browser (CSS de impressão)
- [ ] AC-5: Exportação como PDF (via biblioteca jsPDF ou similar)
- [ ] AC-6: Calendário associado ao paciente e datado (consulta específica)
- [ ] AC-7: Formato visual amigável para o paciente (ícones de sol/lua/etc.)

## Escopo

**IN:**
- Geração automática a partir de dados já cadastrados
- Edição de horários
- Impressão e PDF
- Layout visual amigável

**OUT:**
- App mobile com lembretes (v3)
- Sincronização com calendário Google/Apple (v3)

## Dependências

- Story 2.2 (medicamentos em uso com horários)
